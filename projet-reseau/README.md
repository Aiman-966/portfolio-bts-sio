# Projet Réseau – DHCP et DNS

![Réseau](https://img.shields.io/badge/Network-DHCP-red)
![DNS](https://img.shields.io/badge/Network-DNS-blue)

## Objectif
Configurer un réseau simple permettant l'attribution automatique d'adresses IP et la résolution de noms.

## Environnement

- Machines virtuelles
- Réseau interne
- Windows Server / Linux

## Étapes réalisées

1. Création du réseau entre plusieurs machines
2. Configuration du serveur DHCP
3. Attribution automatique des adresses IP
4. Configuration du serveur DNS
5. Test de communication entre les machines

## Commandes utilisées

ipconfig  
ping  
nslookup  

## Résultat

Les machines reçoivent automatiquement une adresse IP et peuvent communiquer grâce au serveur DNS.

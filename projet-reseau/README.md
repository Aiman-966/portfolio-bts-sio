# Projet Réseau – DHCP et DNS

Projet réalisé dans le cadre de ma formation en BTS SIO option SISR.

## Objectif

Configurer un réseau simple permettant l'attribution automatique d'adresses IP et la résolution de noms grâce aux services DHCP et DNS.

## Environnement

- Machines virtuelles
- Windows Server / Linux
- Réseau interne

## Étapes réalisées

1. Création d'un réseau entre plusieurs machines virtuelles
2. Installation et configuration du serveur DHCP
3. Attribution automatique des adresses IP aux clients
4. Installation et configuration du serveur DNS
5. Test de résolution de nom entre les machines

## Commandes utilisées

ipconfig  
ping  
nslookup

## Résultat

Les machines du réseau reçoivent automatiquement une adresse IP via DHCP et peuvent communiquer entre elles en utilisant les noms grâce au serveur DNS.
